## Get started
0. Register on https://www.algolia.com

1. Replace 'Application_ID', 'Search-Only_API_Key'

2. Import Dataset (.JSON) provided.

3. You can run this project locally:

```sh
npm install
npm start
```

Visit http://localhost:3000.
